What we cover

The basics that actually stop attacks.

Identity & Access

  • MFA, admin accounts, least privilege, shared account cleanup
  • Secure sharing defaults and external access controls
  • Baseline configuration for M365 / Google Workspace

Hardening & Patching

  • Practical device and account hardening (maintainable)
  • Patch cadence that actually happens
  • Reduce risky defaults and shadow admin access

Vulnerability Management

  • Simple scan/report cadence (where applicable)
  • Risk-based prioritization and tracking
  • Exception documentation when needed

Logging & Visibility

  • Make sure critical events are logged and retrievable
  • Basic high-signal alerts (not noise)
  • Quick “can we investigate?” test

Incident Readiness

  • Simple playbooks and roles
  • Evidence preservation basics
  • Tabletop exercise option

Security Cleanup Sprint

  • Remove obvious exposures fast
  • Lock down accounts/devices that drifted
  • Document changes so you keep the gains
